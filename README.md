# Roll30

A local collection of Dungeons & Dragons 5th Edition SRD data. 🎲

## About

This project holds the full D&D 5e reference data (spells, monsters, classes,
items, and more) as JSON files, downloaded from the open
[5e-bits/5e-database](https://github.com/5e-bits/5e-database) project.

## What's inside

- `data/` — 25 JSON files with all the SRD content (~3.8 MB)

## Data source

Data from the [D&D 5e API](https://www.dnd5eapi.co) / 5e-database, based on the
D&D 5e System Reference Document.
